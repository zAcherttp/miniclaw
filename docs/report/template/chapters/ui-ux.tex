\chapter{Sơ đồ màn hình và tương tác (UI/UX Design)}
\label{chap:ui-ux}

\section{Yêu cầu thiết kế giao diện (Telegram Bot và Command Line Interface)}
Giao diện của Miniclaw được chia làm hai khu vực:
\begin{itemize}
  \item Giao diện hội thoại qua Telegram: Là kênh tương tác chính của người dùng thường nhật, yêu cầu hiển thị ngắn gọn, thân thiện, và tích hợp các nút bấm phản hồi nhanh.
  \item Giao diện dòng lệnh (CLI): Phục vụ cấu hình, onboarding và khởi tạo hệ thống cho nhà phát triển.
\end{itemize}
